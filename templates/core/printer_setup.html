Add the EmberExpress Printer

EmberExpress is a shared network printer backed by our Konica Minolta C4070 — a production-grade color press that most offices could only dream of. Add it once and get crisp, professional 8.5×11 prints in color or B&W, simplex or duplex, right from your own computer.

You must be on the BHMTech Wi-Fi network.
{# ── Windows ── #}
  1. 1

    Open Settings

    Press Win + I, then go to Bluetooth & devices → Printers & scanners.

  2. 2

    Add a printer

    Click Add device. Windows will scan the network. Look for EmberExpress in the list and click Add device next to it.

  3. 3

    Not showing up?

    Click Add manually, choose Add a printer using an IP address or hostname, and enter:

    Hostname / IP: 10.10.96.138

    Protocol: IPP

    Queue: printers/EmberExpress

  4. 4

    Print a test page

    Right-click EmberExpress in your printer list → Printer properties → Print Test Page.

{# ── macOS ── #}
  1. 1

    Open System Settings

    Click the Apple menu → System Settings → Printers & Scanners.

  2. 2

    Add a printer

    Click Add Printer, Scanner or Fax… The Default tab lists nearby Bonjour printers. Click EmberExpress then Add.

  3. 3

    Not showing up?

    Click the IP tab in the Add Printer dialog and enter:

    Protocol: IPP

    Address: 10.10.96.138

    Queue: printers/EmberExpress

    Set Use to Generic PostScript Printer and click Add.

  4. 4

    Print a test page

    Open any document, press ⌘ + P, choose EmberExpress, and print.

{# ── Reminder ── #}
Printer only visible on the BHMTech network. If you don’t see it, check your Wi-Fi first.
